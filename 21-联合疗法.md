# Ch 21 · 为什么未来一定是联合疗法

> 单药打不赢乙肝,已经是行业共识。

---

## 从 HIV 学到的教训

1996 年之前,HIV 单药方案失败率极高。病毒变异太快,任何一种药,几个月就耐药。

那年,"**鸡尾酒疗法**"(HAART)横空出世——**三种药同时用,病毒来不及同时变异应对**。

HIV 从此从"绝症"变成"慢性病"。

**HBV 现在正在走同一条路。**

---

## 为什么单药做不到

回顾第 2 章的病毒生命周期,你会发现——**HBV 有 7 个可以打的环节**。但每种药只打其中一个:

- 核苷药 → 逆转录
- 干扰素 → 转录 + 免疫
- siRNA/ASO → mRNA
- Capsid 抑制剂 → 装配
- Entry 抑制剂 → 入胞
- 治疗性疫苗 → 免疫
- CRISPR → cccDNA

**任何单一环节都有"绕过通道":**

- 核苷药压住复制,但 cccDNA 还在转录 HBsAg
- siRNA 降 HBsAg,但停药后 mRNA 又反弹
- 免疫疗法激活 T 细胞,但抗原压力太大又"耗竭"回去

**只有同时打多个环节 + 唤醒免疫,才能实现真正的"停药治愈"。**

---

## 联合疗法的三种设计思路

**A. 多点抗病毒**
- NUC + siRNA/ASO + Capsid 抑制剂
- 目标:全方位压制病毒,减少抗原
- 特点:**"闷杀"**

**B. 抗病毒 + 免疫调节**
- NUC + siRNA + 治疗性疫苗/PD-1
- 目标:降抗原压力,唤醒免疫
- 特点:**"卸压 + 唤醒"**

**C. 全景轰炸**
- NUC + siRNA + Capsid + 免疫
- 目标:多路夹击
- 特点:**"鸡尾酒"**

**目前主流试验都是 B 或 C 组合。**

---

## 为什么"抗病毒 + 免疫"是核心

单纯把病毒压得再狠,只要**免疫系统还在耐受状态**,停药就反弹。

要真正治愈,必须——

> **在降低抗原压力的同时,让 T 细胞重新识别 HBV。**

这就像给一个疲惫的士兵休息 + 战术训练。休息(降抗原)必须在前,训练(免疫激活)才有效。

**这就是为什么最有希望的方案通常是:**

1. **先** NUC + siRNA/ASO —— 把 HBsAg 打下去(4-24 周)
2. **然后** 治疗性疫苗 / PD-1 —— 教免疫系统重新识别(24-48 周)
3. **最后** 停药,免疫系统接管长期控制

- 三药联合,评估 HBsAg 深度下降

**Vir Biotechnology:VIR-2218 + VIR-3434(中和抗体) + Peg-IFN**

- 抗原降 + 免疫训练

**多家公司 Capsid 抑制剂 + siRNA + NUC**

**详细进展待用 ClinicalTrials.gov 核对最新入组和结果。**

## 一个理想化路径

如果我们把"梦想方案"画出来:

```
Phase 1 (0-12 周):
  NUC + siRNA/ASO 打底
  → HBV DNA + HBsAg 双下降

Phase 2 (12-36 周):
  + Capsid 抑制剂
  → 补货切断,cccDNA 池缓慢缩减

Phase 3 (36-48 周):
  + 治疗性疫苗 or PD-1
  → 免疫系统重新训练

Phase 4 (48 周后):
  停药观察
  → 免疫维持,HBsAg 持续阴,达到功能性治愈
```

**目前多个三期试验正在测试类似路径的变体。**

**关键问题:**
- 需要多长时间?12 个月?24 个月?
- 副作用累积是否可控?
- 停药后维持率多高?
- 成本?

---

## 挑战

联合疗法的路上,还有几个大坑:

**① 副作用叠加**
- 多药同用,不良反应可能协同放大
- 尤其加入免疫调节后,肝损伤风险要密切监测

**② 试验设计复杂**
- 组合太多,不可能都测
- 每个组合需要 200-1000 人规模
- **药厂之间的合作也是难题**——不同公司的药联合用,商业和知识产权障碍

**③ 成本高**
- 单药已经贵,联合更贵
- **如何让全球 2.5 亿患者用得起,是一个必答题**

**④ 定义"停药成功"**
- 停多久算成功?停 1 年?5 年?终生?
- 如何评估"长期免疫控制"的可靠性

## 中国的角色

联合疗法领域,中国正在快速追赶:

- 恒瑞、正大天晴等本土药厂在做 siRNA 和 Capsid 抑制剂
- 数个国产 III 期项目在推进
- **中国患者基数大,是全球联合疗法试验的重要资源**
- 复旦、北大、上海科技大学等实验室在做基础研究

**未来 5-10 年,可能会有中国主导的联合方案获批。**

## 一个患者视角的建议

如果你是一个已经稳定用核苷药的患者,联合疗法对你意味着:

- **不要停药**——除非在临床试验中
- **关注 ClinicalTrials.gov** 上适合你的试验
- **询问医生**你所在城市有没有招募中的三期试验
- **保持 HBsAg 定量低**——很多试验要求 HBsAg < 3000 或 < 1000

**未来几年,如果一个联合方案获批,能否用上,取决于:**
- 你的基线状态
- 医生的意愿
- 医保覆盖
- 药物可及性

**准备好这几点,你就是最先受益的那一批人之一。**
